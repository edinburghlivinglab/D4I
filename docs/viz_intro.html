General Issues

Spreadsheets

Pros:

  • Convenient for small tasks
  • Charting facilities are increasingly sophisticated

Cons:

  • Hard to extend, especially for data cleaning and data manipulation
  • Difficult to connect to multiple or dynamic data sources
  • Hard to port solutions between, say, Excel and Google Sheets

Programming

Pros:

  • Very scalable, can be easily integrated into wider projects, apps
  • Generic tools, applicable to many tasks

Cons:

  • Steeper learning curve
  • Still need to invest in specific frameworks

Processing

Pros:

  • Lots of good libraries
  • IDE works well for small tasks

Cons:

  • Difficult to deploy for web
  • Requires use of Processing.js for embedding in web page

JavaScript

Pros:

  • Lots of choice of different libraries, not only D3.js
  • Browser is pervasive, cross-platform framework for presentation
  • Benefits from innovations in visualisation technology, e.g.Â SVG, HTML5
  • Supports interactive UI

Cons:

  • Steep learning curve
  • Browser as IDE less user-friendly
